# Glossary

## Unity

Game development engine with free and paid tiers of access capable of creating 2D and 3D games, and compatible with third-party plug-ins

## Unity Hub

GUI for managing Unity Installs, repository management, and add-on installation

## Microsoft Visual Studio

Programming IDE created by Microsoft that Unity supports direct integration with.
    

## GitHub

Repository management web service for data analytics, software development, and more
    

## GitHub Desktop

GUI interface for GitHub that allows for more fluid management of local and web repositories on GitHub
